📘 Expense Tracker CLI — Python Project

A simple and efficient Command-Line Expense Tracker built using Python.
It allows users to add income/expense entries, view records, generate monthly summaries, export reports to Excel, and create charts for financial insights.

🚀 Features

➕ Add income & expense entries

📄 Store data in CSV

👀 View all entries in a table format

📆 Monthly financial summary

📊 Generate monthly expense charts (Matplotlib)

📤 Export data to Excel (.xlsx)

✏ Edit existing entries

❌ Delete entries

🔁 Persistent storage
